### 3. The "Secret Word" Checker
**Topic:** String Methods & Conditionals  
**Task:** Create a variable `password = "JavaScript123"`.
**Goal:** Write a script that checks:

**Goal:** 1. Add a new property `lastLogin` with a string value.
2. Update the `isAdmin` property to the opposite of its current value using the `!` (NOT) operator.
3. Delete the `email` property.
Log the final object.

### 5. FizzBuzz (The Classic)
**Topic:** Logic & Modulo  
**Task:** Loop from 1 to 30.
**Goal:** * If a number is divisible by 3, log "Fizz".
* If a number is divisible by 5, log "Buzz".
* If divisible by **both** 3 and 5, log "FizzBuzz".
* Otherwise, just log the number.
